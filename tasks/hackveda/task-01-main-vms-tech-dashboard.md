# Task 01: VMS Techs Dashboard

### Overview
- Built a high-performance, real-time Data Visualization Dashboard to eliminate "Data Lag" and reduce decision-making uncertainty for business analysts.

### Features
- **Real-Time Pulse**: 5-second automatic data refresh using `setInterval`.
- **Dynamic Charting**: Interactive Chart.js line chart for revenue trends.
- **KPI Monitoring**: Live cards for Annual Revenue, System Health, and Uptime.
- **Data Resiliency**: Async/await fetcher with local `mockData.json` fallback.
- **Filtering**: Regional segmentation (North, South, East, West).
